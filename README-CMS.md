# Инструкция для менеджера Sanity CMS

Этот файл помогает быстро и единообразно заполнять админку NESTRO.

## 1. Как добавить апартамент

1. Откройте раздел **Апартаменты**.
2. Нажмите **Create new document**.
3. Заполните вкладку **Основное**: название, slug, статус, комплекс, тип апартамента.
4. Заполните вкладку **Параметры**: цену, валюту, гостей, комнаты, площадь, этаж, вид, кровать и формат аренды.
5. Добавьте главное фото и галерею во вкладке **Фото**.
6. Заполните краткое описание во вкладке **Описание**.
7. Отметьте реальные удобства во вкладке **Удобства**.
8. Проверьте локацию и настройки показа на главной.

Пример апартамента:

- Название: Студия с видом на море • Alliance Palace
- Тип: Студия
- Комплекс: Alliance Palace
- Цена от: 120 GEL
- Гостей: 2
- Комнат: 1
- Площадь: 35 м²
- Этаж: 18
- Вид: Вид на море
- Кровать: Двуспальная кровать
- Удобства: Wi-Fi, Кондиционер, Кухня, Балкон, Лифт, Полотенца
- Краткое описание: Светлая студия рядом с морем, подходит для краткосрочного и длительного проживания.
- Галерея: 5–8 лучших фото
- Показывать на главной: да

## 2. Как создать комплекс

1. Откройте раздел **Комплексы**.
2. Создайте новый документ.
3. Заполните название, slug, адрес, район и главное фото.
4. Добавьте короткое описание, если оно нужно для сайта или менеджеров.

Пример комплекса:

- Название: Alliance Palace
- Район: Первая линия / Новый бульвар
- Адрес: Батуми, рядом с морем
- Описание: Комплекс рядом с морем, ресторанами и городской инфраструктурой.

## 3. Как заполнить настройки сайта

1. Откройте **Настройки сайта**.
2. Создайте один документ настроек.
3. Заполните бренд, WhatsApp, контакты, адрес, часы работы и сообщения для кнопок WhatsApp.

Пример настроек сайта:

- Название бренда: NESTRO
- Подзаголовок: Apartments
- WhatsApp: +995558209739
- Адрес: Батуми, Грузия
- Часы работы: Ежедневно 09:00–23:00
- Сообщение WhatsApp для гостей: Здравствуйте! Хочу уточнить доступность апартаментов в Батуми.
- Сообщение WhatsApp для собственников: Здравствуйте! Хочу обсудить передачу апартамента в управление.

## 4. Как заполнить навигацию

1. Откройте раздел **Навигация**.
2. Создайте документ с названием **Главное меню**.
3. Добавьте пункты меню, ссылки и порядок.
4. Если пункт временно не нужен, выключите **Показывать**.

Пример навигации:

- Преимущества → #advantages
- Апартаменты → #apartments
- Отзывы → #reviews
- Контакты → #contacts

## 5. Как заполнить footer

1. Откройте раздел **Подвал сайта**.
2. Создайте один документ.
3. Заполните описание, copyright и быстрые ссылки.
4. Включайте строку разработчика только если она должна отображаться на сайте.

Пример footer:

- Описание: NESTRO Apartments — апартаменты в Батуми для краткосрочного и длительного проживания.
- Copyright: © NESTRO Apartments. Все права защищены.
- Быстрые ссылки: Апартаменты, Отзывы, Контакты

## 6. Как добавить отзыв

1. Откройте раздел **Отзывы**.
2. Создайте новый отзыв.
3. Заполните имя, оценку, текст отзыва и при необходимости страну или город.
4. Включите **Показывать на главной**, если отзыв должен быть на сайте.

Пример отзыва:

- Имя: Anna
- Оценка: 5
- Текст: Чистые апартаменты, удобное заселение и быстрая связь. Всё понравилось.
- Показывать: да

## 7. Как обрабатывать заявки гостей

1. Откройте раздел **Заявки гостей**.
2. Новые заявки приходят со статусом **Новая**.
3. После связи с гостем переведите статус в **Связались**.
4. Если бронирование подтверждено, установите **Забронировано**.
5. Если заявка не актуальна, установите **Отмена**.

Пример заявки гостя:

- Телефон: +995...
- Сообщение: Хочу квартиру на 3 ночи, 2 гостя, желательно с видом на море.
- Статус: Новая

## 8. Как обрабатывать заявки собственников

1. Откройте раздел **Заявки собственников**.
2. Новые заявки приходят со статусом **Новая**.
3. После первого контакта установите **Связались**.
4. Если идет обсуждение условий, установите **В работе**.
5. Если объект не подходит, установите **Отказ**.

Пример заявки собственника:

- Телефон: +995...
- Комплекс: Alliance Palace
- Тип объекта: Студия
- Сообщение: Хочу обсудить управление апартаментом.
- Статус: Новая
